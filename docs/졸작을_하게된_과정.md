#### 💡 이유
- 기존에 만들어둔 프레임을 기반으로 AI가 디자인을 확장/수정할 수 있는 방법 필요

### 2️⃣ Figma MCP 연결 설정

#### 💬 프롬프트
```
Figma MCP 직접 연결 설정 하는거 처음부터 알려줘
```

#### 📝 결과

**Claude Code 설치 (Native Installer - 2026년 추천 방식)**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Figma MCP 서버 연결**
```bash
claude mcp add --transport http figma https://mcp.figma.com/mcp
```

#### 💡 이유
- npm 설치는 deprecated (더 이상 권장하지 않음)
- Native installer가 더 빠르고, 의존성 필요 없고, 자동 업데이트
- Mac M4 환경에서 가장 간편한 설치 방법

### 5️⃣ 작업 방향 확정

#### 💬 프롬프트 (요구사항 정리)
```
1. 작업 우선순위: 모바일 우선, 둘 다 할 거임
2. 참고 프레임 vs 새로 만들 프레임: 기존 프레임 수정/보완
3. 페이지 우선순위: 메인 → 상품 리스트 → 상품 상세 → 구독 → 예약 → 취향 테스트 → 마이페이지
4. 참고 프레임에서 가져올 것: 레이아웃, 카드 스타일
5. 로고/이미지 에셋: 플레이스홀더로 두고 나중에 넣을 예정
```

#### 📝 결정된 방향
- 모바일 퍼스트로 진행
- 기존 프레임 수정/보완 기반
- 기획서에 있는데 프레임에 없는 경우에만 새로 생성
- 모모스커피 BEST 섹션, 프릳츠 구독 카드 스타일 참고
